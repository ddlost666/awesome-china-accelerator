## 一、为什么海外打开 B 站会显示"该视频仅限大陆地区可观看"？

B 站对部分内容做了版权地区限制，主要包括：

- **番剧**：日本动画大多数仅限大陆 IP 观看
- **影视**：购买的影视剧版权通常限定大陆
- **部分纪录片 / 综艺**：同样存在版权地区限制
- **直播**：部分主播限大陆地区观看

**根本原因**：版权方与 B 站签的授权范围仅限中国大陆。
