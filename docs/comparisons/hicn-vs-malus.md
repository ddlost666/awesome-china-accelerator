## 一、两款产品定位差异 · Positioning

### HiCN

> **专为海外华人、留学生及海外差旅人士潜心打造的高端专属回国网络加速器**

- 目标用户清晰：海外华人 / 留学生 / 海外差旅
- 场景覆盖广：游戏 + 视频 + 直播 + 体育 + 音乐 + 办公
- 设计思路：一站式全场景解决方案

### Malus

> 全球回国加速器，主打高清视频与游戏双场景

- 目标用户较泛：海外华人通用
- 场景覆盖：视频 + 游戏为主
- 设计思路：以视频和游戏两大主流场景为核心

**结论**：HiCN 的场景覆盖更全面，Malus 的场景边界更聚焦但也意味着覆盖面更窄。
